# Silver Tier Gmail Watcher - Create Template

## Purpose

This template demonstrates the structure for markdown files created by the Silver Tier Gmail Watcher when processing unread important emails from Gmail.

## Metadata Fields

| Field | Description | Example |
|-------|-------------|---------|
| **from** | Email sender | "John Doe <john@example.com>" |
| **subject** | Email subject line | "Urgent: Project Update Required" |
| **snippet** | First 200 chars of email body | "Hi team, we need to discuss..." |
| **priority** | Priority level (high/normal/low) | "high" |
| **is_important** | Gmail important marker | true/false |
| **email_id** | Gmail message ID | "18f3a2b4c5d6e7f8" |
| **received_date** | When email was received | "Wed, 11 Mar 2026 10:30:00 +0000" |
| **processed_at** | When file was created | ISO 8601 timestamp |
| **tier** | Vault tier level | "Silver" |

## How It Works

1. **Authentication**: Uses OAuth2 with credentials from `AI_Employee_Vault/Silver_Tier/credentials.json.json`
2. **Polling**: Checks Gmail every 5 minutes (configurable) for unread emails
3. **Processing**: Extracts metadata (from, subject, snippet, priority)
4. **Output**: Creates markdown file in `AI_Employee_Vault/Silver_Tier/Needs_Action/`
5. **Priority Detection**: Automatically detects important emails based on:
   - Gmail's important marker
   - Subject keywords (urgent, ASAP, important, action required)
   - Email headers (X-Priority, Importance)

## Usage

### Run Once (Single Check)
```bash
python gmail_watcher.py --once
```

### Run Continuously (Default - 5 min interval)
```bash
python gmail_watcher.py
```

### Custom Interval
```bash
python gmail_watcher.py --interval 600
```

## Files Created

Each processed email creates a markdown file with:
- YAML frontmatter with all metadata
- Structured content with sender, subject, snippet
- Action items checklist
- Status tracking
