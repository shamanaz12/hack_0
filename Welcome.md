# Welcome to Your Obsidian Vault

This is your new Obsidian vault. Here's how to get started:

## Basic Navigation
- Use the left sidebar to explore your files
- Press `Ctrl+O` to open the quick switcher
- Press `Ctrl+P` to open the command palette

## Useful Shortcuts
- `Ctrl+N`: Create new note
- `Ctrl+S`: Save current note
- `Ctrl+F`: Search in current note
- `Ctrl+Shift+F`: Global search

## Linking Notes
To link to another note, use double brackets: [[Another Note]]

## Tags
Add tags to organize your notes: #work #ideas #research

## Next Steps
1. Explore the settings to customize your experience
2. Check out the graph view to visualize connections
3. Try using canvas for visual thinking
4. Experiment with different plugins

Happy note-taking!
