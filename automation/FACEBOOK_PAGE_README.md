# Facebook Business Page Auto-Post

**Page ID:** 61578538607212  
**Page URL:** https://www.facebook.com/profile.php?id=61578538607212

## Quick Commands

### 1. Auto Post to Page
```bash
python facebook_page_auto.py --post "Gold Tier - Your partner in business growth!"
```

### 2. Get Recent Posts
```bash
python facebook_page_auto.py --get
```

### 3. Delete a Post
```bash
python facebook_page_auto.py --delete 1
```

### 4. Check Status
```bash
python facebook_page_auto.py --status
```

## Features

- ✅ Direct page redirect (no login issues)
- ✅ Auto post to business page
- ✅ Get recent posts
- ✅ Delete posts
- ✅ Uses saved session cookies
- ✅ No MCP server dependency

## First Time Setup

1. Run once with visible browser:
   ```bash
   python facebook_page_auto.py --status
   ```

2. Login to Facebook if prompted

3. Session is saved automatically

4. Next runs use saved session (can use --headless)
