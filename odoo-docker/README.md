# Obsidian Vault

This is an Obsidian vault setup. Obsidian is a powerful knowledge management tool that works on top of a local folder of plain text Markdown files.

## Getting Started

1. Install Obsidian from [https://obsidian.md](https://obsidian.md)
2. Open this folder as a vault in Obsidian
3. Start taking notes!

## Folder Structure

- `.obsidian/` - Configuration files for this vault
- `Attachments/` - Images and other media files
- `Daily Notes/` - Daily note entries
- `Templates/` - Template files for new notes

## Features Enabled

- Graph view for visualizing connections
- Daily notes for journaling
- Templates for consistent note creation
- Canvas for visual thinking
- Backlinks and outgoing links for navigation
